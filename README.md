easy-swift-localize
===================
I am a iOS new-bee
And have not found an easily use library for swift localization.
Try this code,it does help me.



HOW TO USE:


1.Drag in the "String+local.swift" in your project
2.Add ".loc" after the stings you want to localize.
BEFORE:
    text = "china"
AFTER:
    text = "china".loc
